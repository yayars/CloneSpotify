# Clone Do Spotify

Este é  um projeto de clone do Spotify, desenvolvido para fins educacionais. O objetivo é criar uma interface semelhante á do Spotify, utilizando tecnologias web modernas.

## Tecnolgia Utilizadas

- HTML5
- CSS3
- Javascript 
- Tailwind CSS (versão 4.x)
- Node.js (versão 24.x)

## Instalação

1. Clone este repositório:

```bash
git clone https://github.com/yayars/CloneSpotify.git
```

2. Navegue até o diretório do projeto:

```bash
cd CloneSpotify
```

3. Instale as dependências:

```bash
npm install
```
